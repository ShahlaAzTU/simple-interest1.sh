# Emotion Detection Application with Watson NLP

Bu layihə IBM Watson Natural Language Processing (NLP) kitabxanasından istifadə edərək mətnlərdə emosiyaları aşkar edən sadə bir Python tətbiqidir.  
Layihə Flask framework vasitəsilə web deployment üçün hazırlanmışdır və istifadəçilərə mətni daxil edərək emosional analiz nəticəsini JSON formatında əldə etməyə imkan verir.

## 📌 Layihənin əsas xüsusiyyətləri
- Watson NLP kitabxanası ilə emosiyaların aşkarlanması
- Flask server üzərindən web interfeys
- Error handling (boş input və status code 400 üçün)
- Unit testlər ilə funksionallığın yoxlanması
- PyLint ilə static code analysis
